What's under the hood

Three layers,
one command.

01

Backend

FrankenPHP runs Symfony in worker mode — persistent PHP processes, zero cold starts. Caddy handles TLS, compression and routing. JWT authentication included.

02

Frontend

Vanilla JS with a curated library stack — Bootstrap, Swiper, SweetAlert2, Jarallax. Vite bundles everything in milliseconds and hot-reloads in dev.

ViteBootstrap 5Vanilla JS
03

Infrastructure

Fully containerized. Isolated Docker bridge network — only the frontend port is exposed in production. Database never touches the public interface.
